## 풀이 코드

```python
import sys
input = sys.stdin.readline
n = int(input())
nums = list(map(int,input().split()))
dp = [0]*n

dp[0] = nums[0]
for i in range(1,n):
    dp[i] = max(dp[i-1]+nums[i],nums[i])

print(max(dp))
```
