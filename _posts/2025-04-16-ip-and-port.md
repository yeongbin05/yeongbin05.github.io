- **IP**: 네트워크상에서 한 컴퓨터를 가리키는 주소
- **Port**: 특정 컴퓨터 내에서 실행 중인 프로그램을 가리키는 주소

포트 번호는 0부터 65535(2의 16승)까지 사용할 수 있습니다. 0번부터 1023번까지는 규약에 따라 정해진 Well-known port입니다.

| 프로토콜 | 포트 |
| --- | ---: |
| HTTP | 80 |
| HTTPS | 443 |
| FTP | 21 |
| SSH | 22 |
