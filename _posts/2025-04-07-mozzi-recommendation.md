사용자의 식습관, 알레르기 정보, 냉장고 식재료 데이터를 활용하여 맞춤형 음식을 추천하는 백엔드 시스템을 구축했습니다.

## 주요 기능

- 사용자별 음식 및 식재료 선호도 분석
- 냉장고 보유 식재료 기반 추천 알고리즘 적용
- RDBMS를 활용한 음식-사용자 관계 데이터 관리

## 기술 스택

- Frontend: React Native, TypeScript
- Backend: Django, Spring
- Database: MySQL, MongoDB
- Infra: Docker, Nginx, AWS EC2

## 맡은 역할

- DB 및 API 설계
- 공공 API에서 받은 데이터 전처리
- 추천 알고리즘 개발
- Django를 이용한 API 개발
